Introduction

Though this article is not written to explain the detail of k-means algo, it is necessary to remember the basic framework. Given a k value, the algo update centroids’ coordinates in a two-step fashion every loop. Loops will be ended if certain number of times are run or convergence condition is satisfied. Both are given by user.

Realization

Features of this application include:

  1. No extra storage needed. After every loop, the intermidiate file will be updated so there is no accumulation in disk consumption.
  2. Every loop the newly updated centroid file is automatically cached into the distributed cache to accelerate every job.
  3. Free choice of convergence condition and K value.
  4. Combiner is used to reduce reducers’ work load.

Some Thoughts

K-means fits MapReduce frame naturally, which makes it much faster to calculate. Yet, no matter how absurd K value is set, the algorithm will always converge. Only through cross validation can we decide a proper K, which cost extra computational resource.
